Essai materi mata kuliah SIG WEB

Secara keseluruhan sudah dibahas bagaimana internet bekerja melalui sistemnya yang rumit, kemudian apa saja komponen yang membangun sebuah website, seperti HTML sebagai elemen tubuh, CSS sebagai penghias atau pengedit tampilan agar lebih menarik dan Javascript yang akan menambahkan action sehingga halaman web lebih interaktif. HTML dan CSS bukan bahasa pemrograman tetapi hanya bahasa MARKUP. Berbeda dengan Javascript yang merupakan bahasa PEMROGRAMAN. Selain itu juga dipelajari tentang penggunaan Bootstrapt untuk halaman web, bisa juga dengan menggunakan template yang ada di web bootstrap. Juga dipelajari penambahan peta ke dalam website dengan cara paling sederhana yaitu menggunakan leaflet. Dilanjutkan dengan penambahan File Geojson format ke dalam website. File vektor bisa diubah menjadi GEOJSON dan bisa ditambahkan ke dalam website.
